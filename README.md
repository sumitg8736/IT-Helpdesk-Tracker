# 🖥️ IT Helpdesk Ticket Tracker

A fully functional IT support ticket management system built with **HTML, CSS, and vanilla JavaScript** — no frameworks, no dependencies.

## 📋 Features

- **Create Tickets** — Log IT issues with title, description, category, priority, and assignee
- **Priority Levels** — Low / Medium / High / Critical with color-coded badges
- **Status Tracking** — Open → In Progress → Resolved → Closed workflow
- **Quick Status Update** — Change ticket status directly from the detail view
- **Search & Filter** — Filter tickets by status, priority, or search by keyword
- **Dashboard Stats** — Live count of total, open, in-progress, and resolved tickets
- **Persistent Storage** — All data saved in browser's localStorage
- **Fully Responsive** — Works on desktop and mobile

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure & semantic markup |
| CSS3 | Custom properties, Grid, Flexbox, animations |
| JavaScript (ES6+) | DOM manipulation, localStorage, dynamic rendering |

## 💼 Real-World Relevance

This project mirrors tools used in **IT Support roles at companies, where:
- Tickets are raised for hardware/software/network issues
- Engineers update ticket status as they work through resolutions
- Priority levels determine response SLAs
- Dashboards give supervisors real-time visibility

## 🔧 How to Run

1. Clone the repository
2. Open `index.html` in any browser
3. No installation or server required!

## 🌐 Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your live URL will be: `https://yourusername.github.io/it-helpdesk-tracker`
